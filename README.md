# Virtual Electrical Engineer

This repository collects the circuit design packages from the workspace, including BOMs, schematics, firmware sketches, and generated PDFs for sharing with collaborators.

Included project folders:

- `aaa_flashlight`
- `circuit_package_too_close_alarm`
- `circuits`
- `high_power_led_arduino_9v`
- `opamp_amplifier_1v_to_5v`
- `rangefinder_package`
- `rfid_solenoid_access`
- `rfid_solenoid_access_v2`
- `safe_lock_circuit`
- `snake_matrix_circuit`
- `sun_tracking_camera`
- `temp_humidity_sd_logger`

The local Python virtual environments and editor caches are intentionally excluded from the repo.

Each package directory is self-contained. Start with the `design_summary.md`, `circuit_summary.md`, or package-specific `README.md`/`bom.md` files in the folder you want to review.
